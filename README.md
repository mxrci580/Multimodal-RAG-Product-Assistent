# Multimodal RAG Product Assistant

An AI-powered Multimodal Retrieval-Augmented Generation (RAG) system that enables users to search and understand products using both text and images.

## Features

- Gemini 2.5 Flash
- Gemini Embeddings
- ChromaDB Vector Database
- Semantic Product Search
- Multimodal Retrieval
- Streamlit Interface

## Tech Stack

- Python
- Google Gemini
- ChromaDB
- LangChain
- Streamlit
